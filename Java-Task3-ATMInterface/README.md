# ATM Interface

# Objective
A console-based ATM Interface developed in Java for Oasis Infobyte Internship.

# Features
- User Login
- Deposit Money
- Withdraw Money
- Check Balance
- Transfer Money
- Transaction History
- Quit

# Technologies Used
- Java
- VS Code

# Project Structure
- Main.java
- ATM.java
- Account.java
- Bank.java
- Transaction.java

# How to Run

Compile:
javac *.java

Run:
java Main
