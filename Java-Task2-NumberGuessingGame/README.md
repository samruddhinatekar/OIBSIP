# Project Description

The Number Guessing Game is a Java Console Application developed as part of the Oasis Infobyte Java Development Internship.

The system generates a random number between 1 and 100, and the player has to guess the correct number within 7 attempts.

The game provides hints such as:
- Too High
- Too Low
- Correct

It also keeps track of:
- Attempts
- Rounds Played
- Rounds Won
- Rounds Lost
